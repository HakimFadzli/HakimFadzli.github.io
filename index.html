Binary Search Trees(BST)

  • Notes from CSC248
  • A binary search tree is a binary tree in which each element resides on a unique path from the root.
  • Definition of BST:
  • A binary search tree (t) is a binary tree such that either(t) is empty or
  1. Each element in leftTree is less than the root element of t.
  2. Each element in rightTree is reater than root element of t.
  3. Both leftTree and rightTree are binary search tree.
